# Store Maintenance Web App
A web application to capture and manage store maintenance requests from field employees.

## Features
- Capture store maintenance details
- Upload images and audio
- Store data in cloud database
- Integration with Google Sheets

## Tech Stack
- Frontend: HTML, JS
- Backend: Node.js (API)
- Deployment: Vercel

## Setup
1. Install dependencies
2. Configure environment variables
3. Run locally or deploy

## Notes
- Do not commit `.env` or service account keys
